Detailed Description

ADOdb Lite is a PHP class to encapsulate multiple database APIs and is compatible with a subset of the ADODB Command Syntax. Currently supports Frontbase, MaxDB, miniSQL, MSSQL, MSSQL Pro, MySQLi, MySQLt, MySQL, PostgresSQL, PostgresSQL64, PostgresSQL7, SqLite and Sybase.
